# Theme Assets

Sostituisci questi file per cambiare la direzione artistica senza toccare la logica.

- `bg-main-default.svg`: sfondo principale full-bleed
- `bg-overlay-default.svg`: bagliori, luci, haze, skyline, silhouette
- `map-fragment-default.svg`: linee, marker, frammenti cartografici
- `grain-default.svg`: texture superficiale
- `icon-seal-default.svg`: sigillo del brand in alto
- `icon-story-default.svg`: icona del prologo
- `icon-mission-default.svg`: icona della missione
- `icon-complete-default.svg`: icona del completamento

Per usare nomi diversi, aggiorna l'oggetto `THEME_SETTINGS` dentro `index.html`.
